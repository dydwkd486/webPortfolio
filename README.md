# webPortfolio

Danggeun Market 무드를 참고한 정적 웹 포트폴리오 템플릿입니다. 빌드 없이 바로 GitHub Pages에 올릴 수 있도록 구성했습니다.

## 구성
- `index.html`: 메인 포트폴리오
- `styles/main.css`: 공통 스타일
- `scripts/site-content.js`: 이름, 소개, 프로젝트 등 편집 데이터

## 사용 방법
1. `scripts/site-content.js`에서 이름, 소개, 링크, 프로젝트 내용을 수정합니다.
2. GitHub 저장소 루트에 그대로 푸시합니다.
3. GitHub Pages를 `Deploy from a branch`로 설정하고 브랜치 루트를 배포하면 됩니다.

## 메모
- 정적 사이트라 별도 설치 없이 열어볼 수 있습니다.
- GitHub Pages 호환성을 위해 상대 경로만 사용했습니다.
